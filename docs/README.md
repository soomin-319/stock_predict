# 문서 안내

`docs/` 루트에는 현재 운영/개발에 자주 참고하는 문서만 둡니다.

## 핵심 문서

- `ARCHITECTURE.md`: 파이프라인 구조와 주요 경계
- `OPERATIONS.md`: 실행, 실데이터 갱신, Kakao/Colab 운영
- `PROJECT_FEATURES_OVERVIEW.md`: 프로젝트 기능 요약
- `PREDICTION_FORMULAS.md`: 예측/피처/신호 공식
- `EXTERNAL_DATA_INTEGRATION_GUIDE.md`: 외부 데이터 연동 가이드
- `ROADMAP.md`: 우선순위와 향후 작업
- `CODEBASE_ANALYSIS.md`: 코드베이스 상세 분석

## 보관 문서

과거 분석 리포트, 개선안, 작업 계획/설계 기록은 `archive/`에 보관합니다.
