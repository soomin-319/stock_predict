# Archive

현재 상시 참고 대상은 아니지만, 이력 확인을 위해 보관하는 문서입니다.

## 구조

- `reference/`: 기존 구조, 운영, 기능, 공식, 연동 참고 문서
- `analysis-reports/`: 날짜별 코드베이스 분석, 개선안, 리뷰 리포트
- `architecture/`: 아키텍처와 프로젝트 구조 문서
- `integrations/`: 외부 데이터 및 서비스 연동 문서
- `operations/`: 실행, 운영, 산출물 관리 문서
- `plans/`: 로드맵과 과거 구현 계획
- `prediction/`: 예측 공식과 모델 관련 문서
- `project-overview/`: 프로젝트 기능 개요 문서
- `specs/`: 과거 설계 문서
